# Snakes and Ladders Game (C)

This project implements a simulation of the classic Snakes and Ladders board game in C.

## Overview

The game simulates a single player moving across a 10x10 board using dice rolls.  
The player interacts with snakes and ladders and continues until reaching the final cell.

## Features

- Random dice rolls
- Board represented as a 2D matrix
- Snake and ladder detection using recursion
- Player movement logic
- Game simulation until completion
- Tracks the total number of turns

## Technologies

- C
- Structs
- Arrays (2D Matrix)
- Recursion

## How to Run

Compile: gcc snakes_and_ladders.c -o game

Run: ./game

## Output

- Dice rolls per turn
- Player movement updates
- Snake and ladder interactions
- Total number of turns at the end of the game
